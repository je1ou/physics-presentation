## Описание
Интерактивная презентация по физике, созданная с использованием Reveal.js. Включает теорию, конвертеры величин и решатель задач.

## Структура проекта
- `index.html` - основная презентация
- `theme.css` - стили оформления
- `script.js` - JavaScript код (конвертеры и решатель)
- `README.md` - этот файл

## Функционал
1. **Теория** - 7 категорий единиц измерения:
   - Скорость
   - Температура
   - Длина
   - Масса
   - Площадь
   - Объём
   - Давление

2. **Конвертеры величин** (4 категории):
   - Температура (°C ↔ °F ↔ K)
   - Длина (м ↔ км ↔ мили ↔ футы)
   - Масса (кг ↔ г ↔ фунты ↔ унции)
   - Скорость (м/с ↔ км/ч ↔ мили/ч ↔ узлы)

3. **Задачи на скорость**:
   - Теория формулы S = v × t
   - Пример решения
   - Интерактивный решатель

4. **Весёлые картинки** - юмористические иллюстрации на тему физики
